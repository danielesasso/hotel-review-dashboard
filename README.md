## üìä Project Overview
This project explores hotel review data through an interactive Tableau dashboard, focusing on:

- Geographic distribution of hotels (heatmap)
- Customer satisfaction across multiple dimensions (radar chart)
- Satisfaction trends over time (line chart)
- Traveler demographic distributions (pie charts)

The dashboard allows users to filter by hotel and review dimensions to compare patterns and trends visually.

---

## üß† Analytical Focus
The main goals of this project were:

- Designing a **clean, interactive dashboard layout**
- Handling **multi-table datasets with imperfect relationships**
- Applying **best practices in Tableau filtering and context**

Because the dataset is synthetic, the analysis emphasizes **relative comparisons and visual insights** rather than absolute real-world conclusions.
